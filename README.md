# PR Reviewer Assignment Service

Проект для управления pull request и назначением ревьюверов.

## Требования

- Docker
- Docker Compose
- Go 

## Настройка

# Поднимаем сервисы
make docker-up

# Отключаем сервисы
make docker-down

# Применить все миграции
make migration-up

# Откатить миграции
make migration-down
